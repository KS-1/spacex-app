SpaceX Angular App

This simple web app was built using Angular5 and uses the SpaceX Data REST API to provide some simple information about the SpaceX Company, Rocket Fleet, and Launches. The site will continue to grow and improve over time, you can click the buttons below to view the information on the site, or you can check out the links at the bottom of the site to see the technologies used to build it.
